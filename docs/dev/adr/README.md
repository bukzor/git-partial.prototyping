# Architecture Decision Records

Date-based ADRs: `YYYY-MM-DD-NNN-title.md`

## Quick Search

```bash
# Recent ADRs
ls -1t docs/adr/*.md | grep -v README | head -5

# ADRs from specific month
ls docs/adr/2025-11-*.md

# Search by keyword
grep -l "keyword" docs/adr/*.md
```
